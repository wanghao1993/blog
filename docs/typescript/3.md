# interface

接口通常用于声明对象类型的一种方式

例如：

```typescript
interface Person {
    name: string,
    age: number
}

const a: Person = {
    name: 'xx',
    age: 11
}

function printPerson(p: Person) {
    console.log(p.name, p.age)
}

printPerson(a)

```

此外Ts支持可选属性和联合属性
