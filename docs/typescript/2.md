# 函数

## 函数声明
声明一个函数，由函数主体，参数值和返回值组成
```typescript
function test(s: string): string {
    return s
}

// 参数t？表示参数可选
function paramsIsNotNecessary(s: string, t?: string): string {
    return s
}

// 参数t=xx表示参数t的默认值
function paramsIsNotNecessary(s: string, t = 'b'): string {
    return s + t
}
```
